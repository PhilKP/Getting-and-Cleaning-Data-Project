## Code Book

This code book describes the variables(columns) in `tidy.txt`, created by the run_analysis() function in the run_analysis.R script file, submitted as part of this project work.

### Identifiers

* **Subject** - The Id of the test subject
* **Activity** - The type of activity being performed at the time of measurements

### Activity Description

| **#**     | **Activities**           		 | **Description**													|
| :---: |:---------------------------|:-------------------------------------------------------------|
| 1 	| WALKING 					 | Subject was walking during the test 							|
| 2 	| WALKING_UPSTAIRS       	 | Subject was walking up a staircase during the test 			|
| 3 	| WALKING_DOWNSTAIRS      	 | Subject was walking down a staircase during the test 		|
| 4 	| SITTING      	 			 | Subject was sitting during the test 							|
| 5 	| STANDING  		      	 | Subject was standing during the test							|
| 6 	| LAYING 			      	 | Subject was laying down during the test						|
